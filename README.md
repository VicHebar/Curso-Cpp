# Curso C++

Este documento es una guía para seguir el curso que se irá subiendo en esta plataforma, se irá actualizando constnatemente, yo daré aviso cuando se actualice.

## Acerca del curso.

Esta será la manera de dar seguimiento a nuestro curso de C++, a manera de instrucciones dejaré esta guía para saber cómo usar la plataforma, las carpetas con las instrucciones llevaran cierta nomenclatura que de momento no he decidido, pero para ser practicos y hasta nuevo aviso será ```cp_enumeracion```, en caso de ser una carpeta para especificar alguna tarea o actividad será marcada com ```actividad_algunNombre```, es sencilla y no creo tener que explicarla, simplemente sigan la enumeración y podrán llevar el curso en orden.

## Carpetas.

Las carpetas tendrán el siguiente contenido:  
1. Un archivo .md
2. Un archivo makefile
3. Carpeta ```src```
4. Carpeta ```include```
5. Carpeta ```bin```
6. Puede ser que ocacionalmente se encuentren archivos adicionales.

A continuación se explicara la lista anterior.

### Archivo .md

Los archivos .md (a partir de este punto me referiré por ellos como archivos markdown) son documentos de texto que no pierden formato, es decir, se pueden abrir con cualquier editor de texto y aún así se podrán leer. Un ejemplo de lo que intento reflejar sería *Word* este software nos permite editar texto con formato en archivos .docx, sin embargo, si el día de mañana desapareciera el programa *Word* no se podría volver a ver un documento .docx de manera legible para un hmano. Para más información de markdown recomiendo la [siguiente página](https://markdown.es/sintaxis-markdown/), como prueba de lo fácil de usar, este archivo es un archivo markdown.

### Archivo makefile

Es el archivo que usamos para compilar nuestras aplicaciones y programas, en algunas ocasiones los archivos makefile harán más que compilar el programa, en dados casos dentro del archivo markdown vendrán sus instrucciones de uso, para más insformación sobre los archivos makefile sugiero dirigirse a [su página](https://www.gnu.org/software/make/manual/make.html).

### Carpeta src

El nombre de esta carpeta viene de source y como du nombre lo indica guarda todo el código fuente del programa o aplicación en cuestión, dentro de él encontrarán los archivos .cpp que contienen (haciendo énfasis) todo el codigo fuente de nuestra aplicaión o programa.

### Carpeta include

En esta carpeta podremos encontrar todos los arachivos de cabecera necesarios para compilar nuestra aplicación, si al leer esto no se entiende el contenido de la carpeta, ¡¡No te preocupes!! al ir avanzando en el curso iremos entendiendo más al respecto.

### Carpeta bin

Esta carpeta siempre etará vacía (gg lol), sin embargo es esencial, ya que en ella irán los archivos compilados, esto se dejará en claro en los arachivos makefile, también veremos eso posteriormente. Es MUY INPORTANTE NO BORRAR DICHA CARPETA ya que el arachivo makefile no podrá hacer su trabajo de manera correcta.

### Archivos y carpetas adicionales

En caso de ser necesario se incluirán archivos y/o carpetas necesarias para el tema en cuestión, en dicho caso el archivo markdown especificará su uso e importancia.

## Resumen

Es importante que sigan las instrucciones de los archivos Markdown, ya que en ellos se especificarán los pasos a seguir, instrucciones, y en general tod el curso, así que en resumen LEAN LOS ARCHIVOS MARKDOWN.
